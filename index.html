Aide

Cette aplication Web vous permet de lire des podcasts XML très facilement. Pour charger un podcast vous devez rentrer son lien dans la textbox et le charger. Celui-ci apparraitra donc dans le menu à gauche. Vous pouvez ensuite y accèder en cliquant dessus. Une fois ceci fait vous verrez la liste des médias que contient le podcast, lors de l'appuie sur "Lire", le média choisis sera chargé dans le lecteur. Si le bouton "Lire" est gris, c'est que vous avez déja lu ce média.
Les podcasts chargés sont automatiquement sauvegardé pour les prochaines utilisations de l'application. Vous pouvez enlever un podcast en cliquant sur le bouton "Supprimer" en dessous de celui-ci. Vous pouvez aussi supprimer tous les podcasts chargés en cliquant sur "Effacer les podcasts".
